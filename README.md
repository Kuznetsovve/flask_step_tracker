# Шагомер
Структура:
- step_counter_kve - папка с проектом:
  - app.py - основной файл приложения
  - config.py - конфигурационный файл (отсутствует в репозитории, содержит секретные переменные)
  - templates:
    - home_page.html - стартовая страница
    - registration.html - форма регистрации
    - login.html - форма авторизации
    - add_step.html - форма ввода количества шагов
    - profile.html - личный кабинет пользователя
    - me.html - информация о пользователе
    - stats.html - страница отображения статистики

  Для отображения результатов за определённый период необходимо в адресной строке после названия периода ввести необходимый перион (например, `./me/month/11`)
